This folder contains four cognitive tutor problems for fraction addition:
1-3plus2-3
1-3plus2-5
1-4plus1-4
1-4plus3-16

It also contains a production rules file in CognitiveModel/productionRules.pr.


The interface is named fractionAddition-mx and is contained in the parent directory.

To run cognitive tutors in Flash, you need to run the Behavior Recorder in "Cognitive Tutor (Jess)" mode.
